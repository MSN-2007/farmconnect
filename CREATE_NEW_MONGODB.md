# Create New MongoDB Atlas - Step by Step

## Step 1: Create New Cluster (5 minutes)

1. Go to https://cloud.mongodb.com
2. Click **"Database"** in left sidebar
3. Click **"Create"** or **"Build a Database"**
4. Choose **FREE** tier (M0)
5. Select a cloud provider (AWS recommended)
6. Select a region (choose closest to you or India)
7. Cluster Name: `FarmConnect` (or keep default)
8. Click **"Create Deployment"**
9. **Wait 3-5 minutes** for cluster to be created

## Step 2: Create Database User (2 minutes)

After cluster is created, you'll see a popup:

1. **Security Quickstart** will appear
2. Choose **"Username and Password"**
3. Username: `farmconnect_admin`
4. Click **"Autogenerate Secure Password"**
5. **COPY THE PASSWORD** - you'll need it!
6. Click **"Create User"**

## Step 3: Add IP Whitelist (1 minute)

1. In the same popup, or go to **"Network Access"**
2. Click **"Add IP Address"**
3. Choose **"Allow Access from Anywhere"**
4. IP: `0.0.0.0/0`
5. Click **"Add Entry"**

## Step 4: Get Connection String (1 minute)

1. Go to **"Database"** in left sidebar
2. Click **"Connect"** on your cluster
3. Choose **"Connect your application"**
4. Driver: **Node.js**
5. Version: **5.5 or later**
6. Copy the connection string

It will look like:
```
mongodb+srv://farmconnect_admin:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
```

7. Replace `<password>` with the password you copied earlier
8. Add `/farm-connect` before the `?` to specify database name

Final format:
```
mongodb+srv://farmconnect_admin:YOUR_PASSWORD@cluster0.xxxxx.mongodb.net/farm-connect?retryWrites=true&w=majority
```
